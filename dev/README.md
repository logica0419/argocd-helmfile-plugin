# Development Guide

## Workspace Setup

Recommended Text Editor: **VSCode**

Run `sh init.sh` in the workspace root directory and follow the instructions.

## Useful Commands

- `task init`
  - Initialize the workspace
- `task spell`
  - Run spell check
- `task lint`
  - Lint Dockerfile & shell script
- `task build`
  - Build the Dockerfile into image

## Tools

- cspell
  - A spell checker.
- hadolint
  - A Dockerfile linter.
